## General info
This project is simple 2d "shotter" game use flat graphics, called sprites, and don't have three-dimensional geometry. They're drawn to the screen as flat images, and the camera (orthographic camera) has no perspective. Is a well executed game along with the subject preference of the creator.


## Technologies
Project is created with:
* framework LÖVE 11.4
* Lua 5.4.4
* Visual Studio Code 1.70


## Setup
Download the latest version of LÖVE from the [@website](https://love2d.org/), and install it.

## Start game
Download inicialization file `*.love` from [@GitHub Pages](https://github.com/PanchenkoI/Love2d). , then just a run the application. Play game and enjoy
